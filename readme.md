# SimpleImgurUploader

Quick upload image to [imgur.com]() **without** login or register api key.

## Quick start
```csharp
using SimpleImgurUploader;

 var uploader = new ImgurUploader();
 var imageLink = uploader.Upload("your-file-path");
```
